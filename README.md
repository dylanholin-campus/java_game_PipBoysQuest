## Javadoc

La javadoc générée est disponible ici :

- `PipBoysQuest/docs/javadoc/index.html`

Pour la régénérer depuis `PipBoysQuest/` :

```sh
javadoc -d docs/javadoc -sourcepath src -subpackages serenadebird.pipboysquest -encoding UTF-8
```
